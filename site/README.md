# Paolo-Wash Site Vitrine

Landing page lavage auto à domicile.

## Déploiement Rapide (Netlify)

1. Zip le dossier `site/`
2. Drag & drop sur https://app.netlify.com/drop
3. URL live instantanée

## Structure

- `index.html` — Landing page complète
- SimplyBook.me widget — À configurer ligne 102

## Configuration SimplyBook.me

1. Créer compte sur https://simplybook.me
2. Configurer services et créneaux
3. Copier l'URL du widget
4. Remplacer ligne 102 dans `index.html` :

```html
<!-- <script src="https://widget.simplybook.it/v2/widget/widget.js"></script> -->
```
